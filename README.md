# 📖 About

BlindIt is a Chrome Extension built for students who use YouTube for learning.

While YouTube offers thousands of educational videos, it also presents many distractions such as:

- Recommended videos
- Shorts
- Trending videos
- Entertainment content
- Clickbait thumbnails
- Suggested videos unrelated to studies

These distractions often reduce productivity and waste valuable study time.

**BlindIt solves this problem by hiding distracting content and allowing users to focus only on educational videos.**

The goal is to transform YouTube into a distraction-free learning platform.

# 🚀 Installation

1. Clone this repository

```bash
git clone https://github.com/Pradeep-GS/Blind-It
```

2. Open Google Chrome.

3. Navigate to:

```
chrome://extensions
```

4. Enable **Developer Mode**.

5. Click **Load Unpacked**.

6. Select the **BlindIt** project folder.

7. Open YouTube and start learning without distractions.

---

# 🚀 How It Works

1. Install the extension.
2. Open YouTube.
3. BlindIt analyzes the page.
4. Distracting sections are hidden.
5. Only educational content remains visible.
6. Continue learning with fewer interruptions.
